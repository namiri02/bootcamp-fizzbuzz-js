# FizzBuzz

Welcome to your first programming exercise in the bootcamp!

## Aims

- Remember how to program!
- Get familiar with Javascript & Node

## Set Up

### Prerequisites

- Git
- Node.js

### Instructions

1. Fork this repository
2. Clone the repo
3. Open in Visual Studio Code
4. Open the terminal (`Ctrl + double tab /`)
5. Run `node main.js`
